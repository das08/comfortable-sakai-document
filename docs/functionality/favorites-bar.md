# Favorites Bar
Comfortable Sakai adds more Course site tabs to favorites bar.  
By default, up to 20 starred Course tabs will be added.

# Example
**If Sakai LMS sets the maximum number of tabs to be displayed in the favorites bar to 10:**
![Before](img/favbar_before.png)
Only 10 Course tabs are displayed in favorites bar, and the rest will have to be selected from `View All Sites button` button.

However, Comfortable Sakai adds more Course tabs (up to 20 by default):
![After](img/favbar_after.png)
`Sample Lecture 11` and `Sample Lecture 12` are added :)
